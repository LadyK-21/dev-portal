## Props

| Prop name   | Description                                                                    | Required? | Default value | Type      |
| ----------- | ------------------------------------------------------------------------------ | --------- | ------------- | --------- |
| `children`  | The element(s) to render within the inner `Card` component.                    | Yes       | N/A           | ReactNode |
| `className` | A string of one or more class names. Passed to the inner `Card` component.     | No        | undefined     | string    |
| `href`      | The destination of the link. Passed to an inner `MaybeInternalLink` component. | Yes       | undefined     | string    |
